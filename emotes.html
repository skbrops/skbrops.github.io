Código dos Emotes

Emotes normais

Para usar um emote, pegue o seu código e digite-o no chat do canal. Exemplo: .grr1 irá exibir o emote correspondente. Só mudar os números para exibir diferentes emotes.

.grr1 ~ .grr43
.wut1 ~ .wut18
.cute1 ~ .cute13
